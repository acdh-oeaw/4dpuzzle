GIS projects

    Figure 1: Screenshot of the GIS project in ArcMap.


To test the AIS one area of Tell el Daba (F/I) which contains the remains of a palace has been analysed. You can access the 2-dimensional GIS project to explore the archaeological evidence of an Egyptian palace. In this movie you can see how the GIS works with the stratigraphic sequencer for the analysis of stratigraphy ( HMC+ movie).

The archaeological features have been reconstructed in a 3-dimensional GIS to test hypotheses of archaeological interpretation. For example, if we digitally reconstruct walls we can see if our interpretation works if the wall does not interfere with other types of archaeological evidence - or if we have created a reconstruction that is not possible.

2D GIS online

You can access the 2-dimensional GIS project that to explore the archaeological evidence of an Egyptian palace. It is possible to see the evidence according to phases (straten) and types of evidence (e.g. tombs).
